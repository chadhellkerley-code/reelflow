Instrucciones de Eliminacion de Datos

Ultima actualizacion: 31 de mayo de 2026

El usuario puede solicitar la eliminacion de sus datos personales y la desconexion de sus cuentas sociales asociadas a ReelFlow siguiendo estas instrucciones.

1. Revocar acceso desde Meta o Instagram

Si conectaste una cuenta de Instagram mediante Meta, podes revocar el acceso desde la configuracion de tu cuenta de Meta o Instagram, en la seccion de aplicaciones, sitios web o integraciones conectadas.

2. Solicitar eliminacion a ReelFlow

Para pedir la eliminacion de datos almacenados por ReelFlow, envia un email a privacidad@reelflow.app con el asunto "Eliminacion de datos ReelFlow".

Inclui en el mensaje la informacion necesaria para identificar tu cuenta, como email de registro, nombre de usuario conectado y plataforma relacionada. No envies contrasenas ni tokens de acceso.

3. Datos que eliminaremos

  • Datos de perfil asociados a tu uso de ReelFlow.
  • Tokens de acceso y autorizaciones guardadas para cuentas conectadas.
  • Borradores, publicaciones programadas e historial interno cuando corresponda.
  • Archivos cargados por el usuario que ya no sean necesarios para prestar el servicio.

4. Plazo de procesamiento

Procesaremos las solicitudes razonables de eliminacion dentro de un plazo de hasta 30 dias, salvo que exista una obligacion legal, tecnica o de seguridad que requiera conservar cierta informacion por mas tiempo.

5. Confirmacion

Una vez completada la eliminacion, enviaremos una confirmacion al email desde el cual se realizo la solicitud o al email asociado a la cuenta.

6. Conservacion limitada

Podemos conservar registros minimos cuando sea necesario para cumplir obligaciones legales, prevenir abuso, resolver disputas o mantener la seguridad del servicio.

Politica de Privacidad

Ultima actualizacion: 31 de mayo de 2026

ReelFlow es una aplicacion para conectar cuentas de redes sociales, cargar videos, preparar publicaciones y programar contenido. Esta politica explica que informacion recopilamos, como la usamos y que opciones tienen los usuarios.

1. Informacion que recopilamos

Podemos recopilar informacion que el usuario entrega directamente al usar la aplicacion, incluyendo nombre, email, configuraciones de cuenta, videos cargados, textos de publicaciones, fechas de programacion y preferencias de publicacion.

Cuando el usuario conecta una cuenta de Instagram, TikTok u otra plataforma, podemos recibir identificadores de cuenta, nombre de usuario, tokens de acceso, permisos autorizados y datos necesarios para publicar o administrar contenido segun la autorizacion otorgada por el usuario.

2. Como usamos la informacion

  • Conectar y administrar cuentas sociales autorizadas por el usuario.
  • Permitir la carga, edicion, programacion y publicacion de videos.
  • Mostrar el estado de publicaciones, borradores, cuentas conectadas e historial.
  • Mejorar la seguridad, estabilidad y funcionamiento de la aplicacion.
  • Cumplir requisitos tecnicos, legales y de las plataformas integradas.

3. Datos de plataformas externas

ReelFlow solo solicita permisos necesarios para las funciones ofrecidas. El usuario puede revocar el acceso desde la configuracion de la plataforma externa correspondiente, como Meta, Instagram o TikTok.

4. Almacenamiento y seguridad

Aplicamos medidas razonables para proteger la informacion del usuario, incluyendo control de acceso a credenciales y tokens. Ningun sistema es completamente seguro, por lo que recomendamos no subir contenido o informacion que no se quiera procesar dentro de la aplicacion.

5. Comparticion de datos

No vendemos informacion personal. Podemos compartir datos con proveedores tecnicos necesarios para operar la aplicacion, como servicios de hosting, almacenamiento, procesamiento de video, analitica, APIs de redes sociales o herramientas de seguridad.

6. Retencion de datos

Conservamos la informacion mientras sea necesaria para prestar el servicio, cumplir obligaciones legales, resolver disputas o mantener registros de funcionamiento. El usuario puede solicitar la eliminacion de sus datos segun se indica en nuestra pagina de eliminacion de datos.

7. Derechos del usuario

El usuario puede solicitar acceso, correccion o eliminacion de sus datos personales. Tambien puede desconectar sus cuentas sociales desde la aplicacion o desde la plataforma externa correspondiente.

8. Menores de edad

ReelFlow no esta dirigida a menores de 13 anos. Si detectamos informacion de un menor sin autorizacion valida, tomaremos medidas para eliminarla.

9. Cambios en esta politica

Podemos actualizar esta politica cuando cambien nuestras funciones, integraciones o requisitos legales. Publicaremos la version actualizada en esta misma pagina.

10. Contacto

Para consultas de privacidad o solicitudes relacionadas con datos personales, escribinos a privacidad@reelflow.app.
